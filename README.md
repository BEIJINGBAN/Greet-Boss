# Boss 打招呼工具

这是一个专门为 Boss 直聘等招聘平台设计的自动打招呼辅助工具。它采用了 **Windows 底层 API 强制切换输入法** 技术，彻底解决了中文输入法干扰网址和字母输入的问题。

---

## 🌟 核心特性

* **⚡ 触发即切英文 (核心优化)**：按下快捷键的瞬间，程序通过 SendMessage 调用 Windows 底层指令，将目标窗口强制切换为 **美国英语键盘布局 (0x409)**。
* **📂 双框预览机制**：展示当前生效配置与实时编辑区，确保所见即所得。
* **⌨️ 捕获式快捷键**：点击“刷新快捷键”后直接在键盘按下组合键，松开即自动保存。
* **🔒 逻辑锁死保护**：模拟真实人工敲击（0.02s 间隔），严格遵循行间延迟。
* **📌 窗口最前**：工具界面始终置顶，方便在聊天窗口快速调取。

---

## 🚀 快速开始

### 1. 环境准备
确保你的电脑安装了 Python 3.x，并安装必要的依赖库：
```bash
pip install pynput
```
### 2. 运行程序
将代码保存为 test.py，在终端或命令行执行：
```bash
python test.py
```
### 3. 标准操作流程
1. **编辑内容**：在右侧 **“内容编辑区”** 输入你的打招呼话术（每一行代表一条独立的消息）。
2. **设定延时**：在 **“延时(s)”** 处输入每条消息发送后的等待时间（建议 1.5s 以上以模拟真人）。
3. **同步配置**：点击 **💾 保存编辑** 按钮。此时左侧只读框会刷新，表示配置已成功写入 JSON 文件。
4. **准备就绪**：点击 **准备开始自动打招呼**，状态栏变为绿色并显示当前监听的快捷键。
5. **一键触发**：点击进入 Boss 聊天框，按下快捷键。
   * 程序将瞬间强制切换输入法为英文，并开始全自动模拟人工输入。

## 🛠 技术实现详情

| 功能模块 | 实现技术 | 逻辑说明 |
| :--- | :--- | :--- |
| **输入法控制** | ctypes + user32.dll | 向活动窗口发送 0x0050 消息，强制加载 0x409 布局 |
| **按键模拟** | pynput.keyboard | 采用 type() 逐字输入而非粘贴，规避平台脚本检测 |
| **配置持久化** | json 异步存取 | 自动创建并读取 boss_config.json，重启依然保留设置 |
| **多线程架构** | threading | 发送逻辑在子线程运行，确保 GUI 界面不会假死或无响应 |

## ⚠️ 常见问题说明

* **关于切换英文**：本功能通过 Windows 底层消息实现，**仅在按下快捷键的那一刻生效**。这确保了输入网址（包含 http/www）时不会弹出拼音候选框。
* **强制置顶**：程序默认开启 topmost 属性，如果不希望遮挡窗口，可根据需要拖动位置。
* **紧急刹车**：如果发送过程中发现窗口选错或内容有误，请 **立即按下 ESC 键**，程序会终止当前所有发送任务。
